# 0. Introduction

## 0.1 About This Guide
Welcome to the VTC MyPortal Student User Guide.

This guide explains how student users navigate and use key portal features, including login, home, assistant, calendar, profile, activities, news, resources, and account-related quick actions.

## 0.2 Intended Audience
This guide is for:
- Current students using VTC MyPortal
- New student users learning the system for the first time
- Students who need step-by-step reference for specific tasks

## 0.3 Guide Objectives
This guide is designed to help you:
1. Access and navigate the portal confidently
2. Complete common student tasks accurately
3. Understand expected system behavior and messages
4. Troubleshoot common issues before contacting support

## 0.4 Scope of Student Guide
The student guide covers user-facing portal workflows, including:
- Authentication and account actions
- Sidebar navigation and quick menu usage
- Academic and activity-related pages available to students
- Reading announcements and accessing resources

The guide does not cover internal dashboard administration workflows.

## 0.5 How to Use This Guide
Recommended usage approach:
1. Start with Chapter 01 (Login)
2. Continue in sequence for first-time learning
3. Use chapter titles to jump directly to specific tasks later
4. Refer to troubleshooting sections when issues occur

Each chapter includes:
- Purpose and context
- Step-by-step instructions
- Expected outcomes
- Troubleshooting notes
- Support handoff details

## 0.6 Visual Placeholders
Image placeholders are included throughout this guide.

They are provided so screenshots can be inserted later for PDF publication.

Example placeholder format:
- Section screenshot
- Form screenshot
- Error message screenshot

## 0.7 Conventions Used in This Guide
To keep instructions clear and consistent:
- Numbered steps indicate exact action order
- Field and button names match UI labels
- Notes and troubleshooting are separated for quick scanning
- Role-specific visibility behavior is stated when relevant

## 0.8 Security and Good Practice Reminder
When using MyPortal:
- Keep account credentials private
- Change password when needed or after potential exposure
- Sign out on shared devices
- Verify important dates and details before taking action

## 0.9 Need Help?
If you cannot complete a task after following chapter instructions, contact support and include:
- Student ID/username
- Page or chapter name
- Action attempted
- Error message (if any)
- Screenshot and device/browser details

## 0.10 Version and Maintenance
This guide should be reviewed whenever portal behavior, labels, or workflows change.

Maintaining alignment between the guide and system UI ensures accurate user support and training.
